## 📋 数据结构详解

### 1. planet_house_readings_sample.csv (行星宫位解读)

**用途**: 当行运行星落入某个宫位时，查询对应的运势解读

**字段说明**:

| 字段 | 类型 | 示例 | 说明 |
|-----|------|------|------|
| id | INT | 1 | 唯一标识 |
| planet | STRING | 木星 | 行星名称 |
| house | INT | 3 | 宫位(1-12) |
| sign | STRING | 双子座 | 行星落入的星座 |
| keywords | STRING | 短途旅行,兄弟姐妹 | 关键词标签 |
| reading | TEXT | 木星在你的第三宫... | 完整解读文本 |
| is_positive | BOOL | TRUE | 是否为正面解读 |
| source_month | STRING | 2025-01 | 数据来源月份 |
| source_sign | STRING | 白羊座 | 数据来源星座 |

**示例数据**:
```csv
id,planet,house,sign,keywords,reading,is_positive,source_month,source_sign
6,木星,3,双子座,"短途旅行,兄弟姐妹,沟通,社交",木星位于你的第三宫...可能会有一次短途旅行。,TRUE,2025-01,白羊座
7,天王星,2,金牛座,"财务,惊喜,逆行结束,资金",天王星...可能会释放你等待已久的资金。,TRUE,2025-01,白羊座
```

**查询示例**:
```python
# 查询: 当木星在3宫时，运势如何？
SELECT reading FROM planet_house_readings 
WHERE planet='木星' AND house=3;

# 结果: "木星位于你的第三宫...可能会有一次短途旅行。"
```

### 2. aspect_readings_sample.csv (相位解读)

**用途**: 当两颗行星形成特定相位时，查询对应的运势解读

**字段说明**:

| 字段 | 类型 | 示例 | 说明 |
|-----|------|------|------|
| id | INT | 1 | 唯一标识 |
| planet_a | STRING | 木星 | 第一颗行星 |
| planet_b | STRING | 太阳 | 第二颗行星 |
| aspect_type | STRING | 三分相 | 相位类型 |
| reading | TEXT | 太阳和新月将收到... | 完整解读文本 |
| is_positive | BOOL | TRUE | 是否为正面相位 |
| source_month | STRING | 2025-01 | 数据来源月份 |
| source_sign | STRING | 白羊座 | 数据来源星座 |

**相位类型说明**:

| 相位 | 角度 | 性质 | 符号 |
|-----|------|------|------|
| 合相 | 0° | 强烈融合 | ☌ |
| 六分相 | 60° | 和谐机会 | ⚹ |
| 四分相 | 90° | 挑战张力 | □ |
| 三分相 | 120° | 顺畅和谐 | △ |
| 对分相 | 180° | 对立平衡 | ☍ |

**示例数据**:
```csv
id,planet_a,planet_b,aspect_type,reading,is_positive,source_month,source_sign
1,土星,太阳,三分相,土星将与太阳和新月呈理想角度...,TRUE,2025-01,白羊座
3,火星,冥王星,对分相,"火星与冥王星对冲已久...",FALSE,2025-01,白羊座
```

**查询示例**:
```python
# 查询: 当火星与冥王星形成对分相时，运势如何？
SELECT reading FROM aspect_readings 
WHERE planet_a='火星' AND planet_b='冥王星' AND aspect_type='对分相';

# 结果: "火星与冥王星对冲已久...你似乎遇到了一两个难缠的上司..."
```

## 💡 使用建议

### 对于开发者

```python
import csv

def load_fortune_data():
    """加载运势数据"""
    data = {
        'planet_house': [],
        'aspects': [],
        'planet_sign': []
    }
    
    with open('planet_house_readings_sample.csv', 'r', encoding='utf-8-sig') as f:
        reader = csv.DictReader(f)
        data['planet_house'] = list(reader)
    
    with open('aspect_readings_sample.csv', 'r', encoding='utf-8-sig') as f:
        reader = csv.DictReader(f)
        data['aspects'] = list(reader)
    
    return data

def get_fortune_reading(planet, house, data):
    """查询运势解读"""
    for row in data['planet_house']:
        if row['planet'] == planet and int(row['house']) == house:
            return row['reading']
    return None
```

### 对于数据维护者

1. **添加新数据**: 按照相同格式追加到CSV文件
2. **去重**: 相同配置只保留最完整的解读
3. **风格一致**: 新数据要保持Susan Miller的温暖鼓励语气
4. **定期备份**: 数据是核心资产，定期备份
