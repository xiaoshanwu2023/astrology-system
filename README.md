# 星盘查询网站使用文档

## 项目概述

这是一个基于 Flask 的星盘查询网站，用户可以输入出生信息，获取个人本命盘、行运盘以及相关的星象解读。

## 功能特性

- ✅ 本命盘计算（太阳、月亮、行星位置）
- ✅ 行运盘计算（当前星象位置）
- ✅ 相位分析（本命盘与行运盘的关系）
- ✅ 星象解读（基于数据库的事件查询）
- ✅ 全国城市选择（34省份，2276城市）
- ✅ 响应式设计（支持移动端和桌面端）

## 技术栈

- **后端**: Python + Flask
- **前端**: HTML5 + CSS3 + JavaScript
- **数据**: Pandas + Excel
- **数据库**: astrology_v2.xlsx（星座事件、行星落入、相位事件）

## 安装和运行

### 1. 环境要求

- Python 3.8+
- pip

### 2. 安装依赖

```bash
cd ~/Desktop/astrology-system
source venv/bin/activate
pip install flask pandas openpyxl pytz
```

### 3. 运行服务

```bash
python app.py
```

服务将在 http://127.0.0.1:5001 启动

## 使用说明

### 1. 访问首页

打开浏览器访问 http://127.0.0.1:5001

### 2. 输入出生信息

- **姓名**（可选）: 输入您的姓名
- **出生日期**（必填）: 选择出生日期
- **出生时间**（必填）: 选择出生时间
- **出生地点**（必填）:
  1. 选择省份
  2. 选择城市（系统会自动加载该省份的城市列表）
- **行运日期**（可选）: 选择要查询运势的日期，默认为今天

### 3. 生成星盘报告

点击"🔮 生成星盘报告"按钮，等待计算完成

### 4. 查看结果

结果页面包含：

#### 本命盘
- 显示出生时各行星的位置
- 包括星座、度数、宫位信息

#### 行运盘
- 显示查询日期各行星的位置
- 显示行星落入的宫位

#### 相位分析
- 本命盘与行运盘的相位关系
- 包括合相、六合、刑相、三合、对冲等

#### 星象解读
- **太阳星座特质**: 基于太阳星座的性格分析
- **上升星座影响**: 基于上升星座的外在表现
- **行星位置指引**: 各行星落入星座和宫位的解读
- **相位运势**: 重要相位的运势解读

### 5. 重新查询或打印

- 点击"🔄 重新查询"返回首页
- 点击"🖨️ 打印报告"打印当前页面

## 数据文件说明

### astrology_v2.xlsx

包含三个工作表：

1. **星座基础事件** (60行)
   - 12星座 × 5类别（性格特征、事业运势、感情运势、财运、健康建议）

2. **行星落入事件** (1440行)
   - 10行星 × 12星座 × 12宫位
   - 覆盖所有行星落入组合

3. **相位事件** (80行)
   - 主要行星组合的相位解读
   - 包括合相、六合、刑相、三合、对冲

### city-data.js

包含全国城市经纬度数据：
- 34个省份
- 2276个城市
- 格式: `{ name: '城市名', coordinates: '经度:分E,纬度:分N' }`

## API接口

### POST /calculate

计算星盘并返回结果

**请求参数**:
```json
{
  "name": "用户姓名",
  "birthDate": "1990-05-15",
  "birthTime": "14:30",
  "coordinates": "116.47,39.90",
  "transitDate": "2024-03-25",
  "transitTime": "10:00"
}
```

**响应**:
```json
{
  "success": true,
  "redirect_url": "/result"
}
```

### GET /result

显示星盘计算结果

## 常见问题

### Q1: 为什么需要准确的出生时间？

出生时间影响上升星座和宫位计算，建议尽量准确。如果不知道确切时间，可以使用中午12:00作为近似值。

### Q2: 城市列表中没有我的出生地怎么办？

选择最近的大城市即可，经纬度的微小差异对星盘计算影响很小。

### Q3: 为什么有些相位没有事件解读？

数据库目前包含主要相位组合的解读，如果某个特定组合没有数据，会显示"暂无解读"。

### Q4: 计算结果可以保存吗？

可以使用浏览器的打印功能将结果保存为PDF。
