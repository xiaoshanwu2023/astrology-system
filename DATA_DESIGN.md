# 星盘事件数据库设计方案

## 设计原则
1. **规范化**: 每种事件类型独立成表
2. **可查询**: 列名清晰，便于程序查询
3. **可扩展**: 方便后续添加新数据
4. **完整性**: 覆盖12星座、主要行星、常见相位

---

## Sheet 1: 星座基础事件 (sign_events)

**用途**: 根据太阳星座或上升星座查询基础性格/运势

| 列名 | 数据类型 | 说明 | 示例 |
|------|----------|------|------|
| sign_zh | 文本 | 星座中文名 | 白羊座 |
| sign_en | 文本 | 星座英文名 | Aries |
| event_category | 文本 | 事件类别 | 性格特征/事业运势/感情运势/财运/健康 |
| event_description | 文本 | 事件描述 | 白羊座的人热情冲动，充满活力... |

**数据量预估**: 12星座 × 5类别 = 60行

---

## Sheet 2: 行星落入事件 (planet_position_events)

**用途**: 根据行星落入的星座和宫位查询事件

| 列名 | 数据类型 | 说明 | 示例 |
|------|----------|------|------|
| planet_zh | 文本 | 行星中文名 | 水星 |
| planet_en | 文本 | 行星英文名 | mercury |
| sign_zh | 文本 | 星座中文名 | 金牛座 |
| house | 整数 | 宫位(1-12) | 2 |
| event_description | 文本 | 事件描述 | 财务思维清晰，善于理财规划... |

**数据量预估**: 
- 重点行星(日月水金火): 5 × 12星座 × 3宫位 = 180行
- 外行星(木土天海冥): 5 × 12星座 × 2宫位 = 120行
- 总计约300行

---

## Sheet 3: 相位事件 (aspect_events)

**用途**: 根据行星间相位查询事件

| 列名 | 数据类型 | 说明 | 示例 |
|------|----------|------|------|
| planet_a_zh | 文本 | 行星A中文名 | 太阳 |
| planet_a_en | 文本 | 行星A英文名 | sun |
| aspect_type | 文本 | 相位类型 | 合相/六合/刑相/三合/对冲 |
| planet_b_zh | 文本 | 行星B中文名 | 月亮 |
| planet_b_en | 文本 | 行星B英文名 | moon |
| event_description | 文本 | 事件描述 | 内心与外在和谐，情感表达顺畅... |

**主要相位说明**:
- 合相 (0°): 能量融合
- 六合 (60°): 机会和谐
- 刑相 (90°): 挑战张力
- 三合 (120°): 天赋才能
- 对冲 (180°): 对立平衡

**数据量预估**:
- 重要行星组合: 10组 × 5相位 = 50行
- 次要组合: 20组 × 3相位 = 60行
- 总计约110行

## 查询逻辑设计

### 场景1: 查询太阳星座事件
```python
query: sign_zh = "白羊座", event_category = "性格特征"
→ 返回对应描述
```

### 场景2: 查询行星落入事件
```python
query: planet_zh = "水星", sign_zh = "金牛座", house = 2
→ 返回对应描述
```

### 场景3: 查询相位事件
```python
query: planet_a_zh = "太阳", aspect_type = "三合", planet_b_zh = "月亮"
→ 返回对应描述
```

### 场景4: 综合查询（实际使用场景）
用户输入出生信息后，系统:
1. 计算本命盘 → 获取太阳星座、上升星座、各行星位置
2. 计算行运盘 → 获取行运行星位置
3. 计算相位 → 获取本命与行运行星间的相位
4. 综合查询:
   - 查询太阳星座基础事件
   - 查询上升星座基础事件
   - 查询各行星落入事件
   - 查询各相位事件
5. 汇总所有事件，按相关性排序展示

## 数据生成策略

### 策略1: 基于现有数据扩展
保留现有4条有效数据，作为模板扩展到其他星座。

### 策略2: 基于占星学原理生成
根据占星学基本原理，为每种组合生成合理的描述:
- 火象星座(白羊/狮子/射手): 强调行动力、热情
- 土象星座(金牛/处女/摩羯): 强调务实、稳定
- 风象星座(双子/天秤/水瓶): 强调沟通、思维
- 水象星座(巨蟹/天蝎/双鱼): 强调情感、直觉

### 策略3: 行星特性 + 星座特性组合
例如:
- 水星(思维) + 金牛座(务实) = 务实思维，善于财务规划
- 火星(行动) + 摩羯座(事业) = 事业心强，追求成就

---

## 数据质量验证

1. **完整性检查**: 确保每个星座、行星、相位都有对应数据
2. **一致性检查**: 确保中英文名称对应正确
3. **唯一性检查**: 确保同一组合没有重复数据
4. **空值检查**: 确保没有空值或无效数据

## 后续扩展计划

1. **添加更多相位组合**: 如梅花相位(150°)、半六分相(30°)
2. **添加宫位事件**: 单独查询某宫位的含义
3. **添加逆行事件**: 行星逆行时的特殊含义
4. **添加时间运势**: 结合具体时间点的运势预测
